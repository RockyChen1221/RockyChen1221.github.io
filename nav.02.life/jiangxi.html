# 三清山

日期: 2017-10-03

这段时间心情有点不好，碰到同学说去爬山看日出，体验了一次说走就走的旅行

# 人物介绍

  • 小G: 同学
  • 小G女友
  • 张大哥一家（偶遇）

确定出发后，自己看了些相关资料，购置了帐篷，登山包、防潮垫、睡袋等等相关道具，没成想爬完这次后就一直闲置吃灰~~

我从上海出发，小G从长沙，他女友从武汉，9月30号晚落地上饶火车站三人碰头，第二天前往山脚

# 萌娃登山篇

选择小道登山时偶遇张大哥一家，问我们那里有寄存婴儿车的地方，我们也不知道，指了指景区的商店，应该放点钱能寄存，玩笑说顺便把娃也给寄了，然后就开始了正式的爬山之旅

中途合影

张大哥一家

边爬边聊，得知碰巧是萌娃出生十个月，他们选择用徒步的方式为宝宝庆祝，工作的地方离我也很近，中途还请我们喝了啤酒，一路上萌娃承担了所有笑点，为我们的旅途增添了许多欢乐色彩

一起爬到了索道的位置，大哥一家因为晚上落脚的原因，无奈做出告别，在这里祝福小宝宝健康快乐成长，大哥一家阖家幸福!开启三人行

傍晚的日落

由于需要赶往搭建帐篷的地点，我们加快了步伐，因为山上黑的比较快，所幸在接近天黑的时候我们到了三清道观（原计划的落脚点），开始了帐篷的搭建，这时候山上已经有了很多人了，能搭帐篷的地方也不多了，而且山上开始转凉了，地上有了露水，短暂收拾后，可算是可以休息了，第二天4点开始起床，前往观日台

# 日出云海篇

伴随着夜里风声、鸟叫、虫鸣、叶子的沙沙声，当然还有鼾声[怒]，以及明月和繁星的夜空离去，迎来了第二天黎明破晓，清晨深山的空气格外清新，用来洗肺不要太好，一望无垠的云海尽收眼底，很是美丽壮观，完整记录了日出的全过程
